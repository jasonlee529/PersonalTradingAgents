## 为什么

当前系统已有涨停池、缠论信号等盘后筛选能力，但缺少一个统一的"量化策略"入口来承载多种技术形态选股策略。个人投研中，"放量回踩"是常见的中继买点形态：先有一波上涨确立趋势，随后回踩均线缩量洗盘，再次放量时往往是继续上行的信号。新增该能力可以让用户从全市场快速筛选出符合该形态的股票，辅助复盘和选股。

## 变更内容

- 新增"量化策略"主页面，以卡片形式列出可用的量化选股策略。
- 实现第一个策略"放量回踩"：最近 20 个交易日上涨 30% 以上 → 回踩 MA10 → 回调过程缩量 → 近期再次放量，并支持策略参数可调。
- 新增后端策略模块 `src/strategies/`，包含策略基类、策略注册表和全市场扫描器，复用现有 `DataCollector` 的行情与 K 线能力。
- 新增后端 API 路由 `/api/strategies`，提供策略列表与策略扫描接口。
- 新增前端策略主页面、API client 方法、路由与侧边导航入口。
- 筛选结果支持按代码/名称搜索、市场筛选，并支持点击跳转到个股行情详情。

## 功能 (Capabilities)

### 新增功能
- `quant-strategy-browse`: 浏览可用量化策略列表。
- `quant-strategy-volume-pullback`: 运行"放量回踩"策略，从全市场筛选出最近 20 天上涨 30% 后回踩 MA10、缩量后再次放量的股票。
